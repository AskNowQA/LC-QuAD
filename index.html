23 Nitpicked Examples
How many rivers start in an English speaking country?

SELECT DISTINCT COUNT(?uri) WHERE { ?x dbo:officialLanguage dbr:English_language . ?uri dbo:sourceCountry ?x . ?uri rdf:type dbo:River . }

Was Reza Amrollahi born in Iran?

ASK WHERE { dbr:Reza_Amrollahi dbp:birthplace dbr:Iran . }

What university campuses are situated in Indiana?

SELECT DISTINCT ?uri WHERE { ?uri dbo:campus dbr:Indiana . ?uri rdf:type dbo:University . }

Who is the person that first ascented the Cathedral Peak and Mount Hoffmann?

SELECT DISTINCT ?uri WHERE { dbr:Cathedral_Peak_(California) dbo:firstAscentPerson ?uri . dbr:Mount_Hoffmann dbo:firstAscentPerson ?uri . }

Which country has given citizenship to both saima chowdhury and edmund john?

SELECT DISTINCT ?uri WHERE { dbr:Saima_Chowdhury dbo:citizenship ?uri . dbr:Edmund_John dbo:citizenship ?uri . }

Who gave the voice for Allen Walker ?

SELECT DISTINCT ?uri WHERE { dbr:Allen_Walker dbo:voice ?uri . }

Who appointed the governor under which Robert Witherspoon worked?

SELECT DISTINCT ?uri WHERE { dbr:Robert_Witherspoon dbo:governor ?x . ?x dbo:appointer ?uri . }

Was Ganymede discovered by Galileo Galilei?

ASK WHERE { dbr:Ganymede_(moon) dbp:discoverer dbr:Galileo_Galilei . }

List the actors of Lucy Sullivan Is Getting Married?

SELECT DISTINCT ?uri WHERE { dbr:Lucy_Sullivan_Is_Getting_Married dbp:starring ?uri . }

Which planet was discovered by Johann Gottfired and Urbain Le Verrier ?

SELECT DISTINCT ?uri WHERE { ?uri dbp:discoverer dbr:Johann_Gottfried_Galle . ?uri dbo:discoverer dbr:Urbain_Le_Verrier . ?uri rdf:type dbo:Planet . }

Where can I find things run by the maritime museum of San Diego?

SELECT DISTINCT ?uri WHERE { ?x dbp:governingBody dbr:Maritime_Museum_of_San_Diego ?x dbp:location ?uri . }

Which labels signs up progressive rock bands?

SELECT DISTINCT ?uri WHERE { ?x dbp:genre dbr:Progressive_rock . ?x dbo:recordLabel ?uri . ?uri rdf:type dbo:MusicalArtist . }

What are some famous works of the writer of The Second Coming?

SELECT DISTINCT ?uri WHERE { dbr:The_Second_Coming_(TV_serial) dbp:writer ?x . ?x dbo:notableworks ?uri . }

Which kind of conventions are held in Rosemont, Illinois?

SELECT DISTINCT ?uri WHERE { ?x dbo:location dbr:Rosemont,_Illinois . ?x dbo:recordLabel ?uri . ?uri rdf:type dbo:Convention . }

Is north america one of the destinations of airtours airways?

ASK WHERE { dbr:Airtours_International_Airways dbo:destination dbr:North_America . }

which politician is currently ruling over Rishkiesh?

SELECT DISTINCT ?uri WHERE { dbr:Rishikesh dbo:leaderName> ?uri . }

Which band's former member are Kevin Jonas and Joe Jonas?

SELECT DISTINCT ?uri WHERE { ?uri dbo:formerBandMember dbr:Kevin_Jonas . ?uri dbp:pastMembers dbr:Joe_Jonas . ?uri rdf:type dbo:Band . }

How many shows does HBO have?

SELECT DISTINCT COUNT(?uri) WHERE { ?uri dbo:company dbr:HBO . ?uri rdf:type dbo:TelevisionShow . }

Where was the designer of REP Parasol born?

SELECT DISTINCT ?uri WHERE { dbr:REP_Parasol dbo:designer ?x . ?x dbo:birthPlace ?uri . }

Name the scientist whose supervisor was Ernest Rutherford and had a doctoral students named Charles Drummond Ellis?

SELECT DISTINCT ?uri WHERE { ?uri dbo:doctoralAdvisor dbr:Ernest_Rutherford . ?uri dbp:pastMembers dbr:Charles_Drummond_Ellis . ?uri rdf:type dbo:Scientist . }

Which company which assembles its cars in Broadmeadows, Victoria?

SELECT DISTINCT COUNT(?uri) WHERE { ?x dbp:assembly dbr:Broadmeadows,_Victoria . ?x dbo:parentCompany ?uri . ?uri rdf:type dbo:Automobile . }

Name some leaders of the parent organisation of the Gestapo?

SELECT DISTINCT ?uri WHERE { dbr:Gestapo dbo:parentOrganisation ?x . ?x dbo:leader ?uri . }

What?

LC-QuAD is a Question Answering dataset with 5000 pairs of question and its corresponding SPARQL query. The target knowledge base is DBpedia, specifically, the April, 2016 version. Please see our paper for details about the dataset creation process and framework.

Every data item in the dataset consists of the following fields:

sparql_template_id: "Every unique SPARQL template has a different ID.",
sparql_query: "Valid SPARQL queries.",
intermediary_question: "The automatically verbalized equivalent of the SPARQL Query.",
corrected_question: "Human corrected version of the verbalized question.",
_id: "Unique ID of the datapoint"
